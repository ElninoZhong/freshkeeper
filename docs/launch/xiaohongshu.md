# 我用一条命令同步 AI 工具 🦞

电脑里装了一堆 AI 编程工具之后，我才发现真正麻烦的不是装，而是更新。  
Claude Code、Codex、OpenClaw、Hermes，每个都要自己管一遍。  
有时候一个月都想不起来更新，等到真要用的时候才发现版本已经落后。  
每次想到要一个个处理，我就先开始假装没看见。

后来我实在被自己拖延症烦到了，就顺手做了一个开源小工具：Freshkeeper。  
它干的事情很简单，就是把这些更新动作收起来，一条命令搞定。  
不用再记每个工具怎么更新，也不用来回翻文档。

如果你也装了不止一个 agent，真的会轻松很多。
